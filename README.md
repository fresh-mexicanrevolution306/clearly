## 📚 Markdown quick guide

Use these common patterns in your text:

- `# Title` for a main heading
- `## Section` for a second-level heading
- `- Item` for a list item
- `**bold**` for bold text
- `*italic*` for italic text
- `[link text](https://example.com)` for a link
